# Screenshots Guide

## üì∏ Required Screenshots

Take the following screenshots and save them in this folder with the exact filenames listed below:

### 1. `main-feed-light.png`
**What to capture:**
- Light mode enabled
- Main page showing at least 2-3 posts in the feed
- Post creation form visible at the top
- Show the header with "Post + Comments" title
- Include at least one post with some comments visible

**How to take it:**
1. Run `npm run dev`
2. Make sure dark mode is OFF (click sun icon if needed)
3. Create 2-3 posts with different content
4. Add a few comments to show the comment count
5. Capture the full page view

---

### 2. `dark-mode.png`
**What to capture:**
- Dark mode enabled (click moon icon)
- Same view as above but in dark theme
- Show how the gradients and colors adapt to dark mode
- Include posts and comments section

**How to take it:**
1. Click the dark mode toggle (moon icon)
2. Ensure you have posts with comments visible
3. Capture the full page in dark theme

---

### 3. `nested-comments.png`
**What to capture:**
- A post with deeply nested comment threads (at least 3 levels)
- Show the visual indentation clearly
- Include both collapsed and expanded threads if possible
- Show the "Reply" buttons and nesting structure

**How to take it:**
1. Click "Add comment" on a post
2. Add a top-level comment
3. Click "Reply" on that comment, add a reply
4. Click "Reply" on the reply, add another nested reply
5. Repeat to show at least 3-4 levels of nesting
6. Capture the entire nested thread

---

### 4. `comment-features.png`
**What to capture:**
- Comment form with emoji picker open, OR
- Comment form with GIF picker open showing the search, OR
- Comment form with image upload preview

**How to take it (Option 1 - Emoji Picker):**
1. Click "Add comment" or "Reply" on any comment
2. Click the emoji icon to open the emoji picker
3. Capture the form with emoji picker visible

**How to take it (Option 2 - GIF Picker):**
1. Click "Add comment" or "Reply"
2. Click the "GIF" button to open GIF picker
3. Type in the search box (e.g., "cat")
4. Capture with GIF results visible

**How to take it (Option 3 - Image Upload):**
1. Click "Add comment" or "Reply"
2. Upload an image using the image icon
3. Capture with image preview visible in the comment form

---

### 5. `post-creation.png`
**What to capture:**
- Post creation form at the top
- Show drag & drop area or uploaded image preview
- Include the "Post" button and text input

**How to take it:**
1. Start typing in the post creation form
2. Upload an image (drag & drop or click upload)
3. Wait for image preview to appear
4. Capture the form with the preview visible

---

## üìê Screenshot Guidelines

- **Resolution:** Capture at 1920x1080 or your native screen resolution
- **Format:** Save as PNG for best quality
- **Cropping:** Try to capture the full browser window (or at least the full content area)
- **Content:** Use meaningful sample text, not Lorem Ipsum
- **Quality:** Ensure text is readable and UI elements are clear

## üéØ Quick Tip

To take clean screenshots:
- Use **Snipping Tool** (Windows) or **Screenshot** (Mac)
- Or press `Win + Shift + S` (Windows) / `Cmd + Shift + 4` (Mac)
- Make sure browser zoom is at 100%
- Hide bookmarks bar if possible for cleaner look
